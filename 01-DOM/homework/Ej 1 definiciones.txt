DOM
El document object model es la estructura HTML de una pagina web. 
Es lo que carga el browser cuando se iniciar y contiene todos los objetos y etiquetas a los cuales se puede acceder con JS.

DOM element selectors
Los selectors son las herramientas que nos permiten identificar y acceder a elementos especificos del DOM 
para poder obtener sus atributos, modificarlos, aplicar eventos, etc.

DOM EVENTS
Los events son scripts que podemos aplicar sobre los elementos del DOM para interactuar y realizar acciones con ellos.
